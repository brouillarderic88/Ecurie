# Module — Soins & Santé

## Objectif
Tracer tous les soins apportés aux chevaux.

## Fonctionnalités
- [ ] Enregistrement d'un soin (date, type, intervenant)
- [ ] Types : ferrure, ostéopathie, dentiste, vétérinaire…
- [ ] Coût associé (pour refacturation éventuelle)
- [ ] Rappels et alertes
