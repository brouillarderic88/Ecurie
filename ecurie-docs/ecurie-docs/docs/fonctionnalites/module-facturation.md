# Module — Facturation

## Objectif
Générer et suivre les factures envoyées aux propriétaires.

## Fonctionnalités
- [ ] Génération automatique de factures mensuelles
- [ ] Postes : pension, soins, cours, divers
- [ ] Envoi par email
- [ ] Suivi des paiements (payé, en attente, en retard)
- [ ] Export comptable
