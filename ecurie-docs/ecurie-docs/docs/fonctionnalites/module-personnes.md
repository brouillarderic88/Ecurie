# Module — Propriétaires & Cavaliers

## Objectif
Gérer les contacts liés aux chevaux (propriétaires, cavaliers, vétérinaires).

## Fonctionnalités

### Fiche contact
- [ ] Nom, prénom, coordonnées
- [ ] Type (propriétaire, cavalier, véto, maréchal…)
- [ ] Chevaux associés
- [ ] Notes

### Espace propriétaire (optionnel)
- [ ] Accès en ligne pour consulter leur cheval
- [ ] Historique des soins visible
- [ ] Factures accessibles

## Questions ouvertes
- Les propriétaires auront-ils un accès à l'application ?
- Un cavalier peut-il être aussi propriétaire ?
